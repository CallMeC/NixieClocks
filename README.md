# Nixie Clocks
This project was made in duo with GregPlusPlus https://github.com/GregPlusPlus !
This is a 4 digit nixie clock project

# Features
- This is able to tell time ! (No jokes)
- Wifi self-hosted webserver for settings
- Buzzer for alarm
- NTP Synchronisation
